C) dongle 用法

C1) dongle 打电话

CLI> dongle cmd dongle0 atd13332900997; 注意最后分号。

C2) dongle 发送 sms

CLI> dongle sms dongle0 +86135608xxxxx hello
CLI> dongle sms dongle0 +8613316874879 abc

C3) dongle 发送 ussd

CLI> dongle ussd dongle0 1161#
CLI> dongle ussd dongle0 *100#

C4) 电话号码写进 sim 卡

To save Subscriber Number, you can write an entry for OWN NUMBERS storage on SIM. For example, in asterisk CLI

CLI> dongle cmd dongle0 AT+CPBS="ON"
CLI> dongle cmd dongle0 AT+CPBW=1,"+85266012433",145

C5) clear SMS, [dongle0] SMS storage is full

CLI> dongle cmd dongle0 AT+CIMI

SMMEMFULL - SMS memory in dongle is full

Clear SMS sim memory

dongle cmd name_dongle AT+CPMS="SM","SM","SM"
dongle cmd name_dongle AT+CMGD=1,4

Clear SMS dongle memory

dongle cmd name_dongle AT+CPMS="ME","ME","ME"
dongle cmd name_dongle AT+CMGD=1,4
