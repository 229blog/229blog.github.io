style>
    h1, h2, h3 { color:blue; }

            }

C) 缩进

&ensp;信仰   缩进半个中文，1个字符
&#8194;信仰  缩进半个中文，1个字符
    
&emsp;信仰   缩进1中文，2个字符
&#8195;信仰  缩进1中文，2个字符
