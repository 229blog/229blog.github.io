Typecho+Apache2安装

Typecho+Apache2安装

1）Apache2 安装

0)typecho.org 官网下载，解压最新软件。
wget http://typecho.org/downloads/1.1-17.10.30-release.tar.gz
1)sudo apt install apache2
2)sudo apt install sqlite3
3)sudo apt install php libapache2-mod-php php-mysql php-sqlite3 php-mbstring php-pdo

2) 之后

1）把typecho.zip移到/var/www/html/ 下。
2）解压typecho.zip.
3）更改文件属性
sudo chmod -R 777 /var/www/html/ 就可以啦！
5)DONE
